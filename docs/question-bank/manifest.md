# Manifest — APTIS General mock question bank

Eight mock sets (`set-01` … `set-08`) for the Premium Language Centre APTIS General platform. All content 100% original, British English, CEFR-calibrated. Import one JSON file per module via **Dashboard → Question bank → Import questions**.

Sets 1–3 are complete (all five modules). **Sets 4–8 cover Core, Reading, Listening and Writing only — no speaking items.** Speaking for those sets is drawn from the sets 1–3 bank.
